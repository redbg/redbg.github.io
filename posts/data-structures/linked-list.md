## Singly Linked List (单向链表)

```cpp
template <class T>
class Node
{
public:
    Node* next;
    T data;
};

template <class T>
class Singly_Linked_List
{
public:
    Node<T>* head;
    unsigned int size;
};
```

```text
        +-----+-----+    +-----+-----+    +-----+-----+    +-----+-----+
head--->|  0  |  •  +--->|  1  |  •  +--->|  2  |  •  +--->|  3  |  ×  |
        +-----+-----+    +-----+-----+    +-----+-----+    +-----+-----+
```

## Singly Circular Linked List (单向循环链表)

```text
        +-----+-----+    +-----+-----+    +-----+-----+    +-----+-----+
head--->|  0  |  •  +--->|  1  |  •  +--->|  2  |  •  +--->|  3  |  •  |
        +-----+-----+    +-----+-----+    +-----+-----+    +-----+--+--+
           ↑                                                        ↓
           +--------------------------------------------------------+
```

## Doubly Linked List (双向链表)

```cpp
template <class T>
class Node
{
public:
    Node* next;
    Node* prev;
    T data;
};

template <class T>
class Doubly_Linked_List
{
public:
    Node<T>* head;
    unsigned int size;
};
```

```text
        +-----+-----+-----+    +-----+-----+-----+    +-----+-----+-----+    +-----+-----+-----+
        |     |     |     +--->|     |     |     +--->|     |     |     +--->|     |     |     |
head--->|  ×  |  0  |  •  |    |  •  |  1  |  •  |    |  •  |  2  |  •  |    |  •  |  3  |  ×  |
        |     |     |     |<---+     |     |     |<---+     |     |     |<---+     |     |     |
        +-----+-----+-----+    +-----+-----+-----+    +-----+-----+-----+    +-----+-----+--+--+
           ↑                                                                                ↓
           +--------------------------------------------------------------------------------+
```
